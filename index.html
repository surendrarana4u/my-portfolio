About Me

I'm a Full Stack Web Developer

I'm a full-stack web developer who enjoys building end-to-end applications that are clean, scalable, and practical. I work across both frontend and backend, focusing on writing clear logic, designing usable interfaces, and delivering features that solve real problems. On the frontend, I build responsive and user-friendly interfaces using modern frameworks and clean UI principles. On the backend, I design APIs, manage databases, and handle authentication and data flow with reliability and performance in mind. I like understanding how things work under the hood—from system design to deployment—so I can build products that don’t just look good, but work well in real-world conditions.
I'm comfortable learning new technologies quickly and improving through hands-on projects, hackathons, and feedback.
Right now, I'm focused on strengthening my full-stack skills, building meaningful projects, and gaining real-world experience through internships and collaborations. I value simplicity, consistency, and writing code that’s easy to maintain.
